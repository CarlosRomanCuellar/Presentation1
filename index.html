Structure

HTML pages are domuments that use tags (also called elements), this tags usually come in pairs to show when an element start and when it ends
the openning one can have attr ibutes that can make the element show or behave one way or another, this attributes require a name and a value.

This tags act like containers that tell you about the information between their opening and closing tags. Examples of this are the html, head and body tags,
The html tag indicates that everything inside is HTML code. The head tag contains information of the page and most of this is not show in the browser
window. The body tag indicates that all between the opening and closing tags is what is going to appear in the browsers window.

Links

Links are the defining feature of the web because with them we move from one web to an other

links are created using the a tag and this tag has the next structure:

<a href="the page the link takes you to">clickable text for the user</a>
A valid example is:
<a href="http://www.google.com">search in google</a>
URL stands for Uniform Resource Locator. This is the web address that you see or type in the browser, when you link to a different website,
the value of the href will be the full web address and this is know as an absolute URL, an example is the one of google that we already see.

Directory

Folders in websites are referred to as directories and is a good idea to organize the code by placing the pages into different folders.
The top level folder is know as root and contains the home page of the site, this page is called index, any folder inside the root is know as a child
and any folder in a child of root is a grandchild of root.

Relative URLs is linking the page with your own website, there are 5 types of relative urls that are:

Type Example
Same folder <a href="examples.html">Examples</a>
child folder <a href="childFolder/examples.html">Examples</a>
Grandchild folder <a href="CFolder/GrandChild/examples.html">Examples</a>
Parent folder <a href="../examples.html">Examples</a>
Grandparent folder <a href="../../examples.html">Examples</a>
